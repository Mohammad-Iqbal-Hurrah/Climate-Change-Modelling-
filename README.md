# Climate Change Modeling

Project repository for exploratory analysis and emission prediction using NASA-related climate datasets.

## Repository Contents
- `Climate_Change_Modelling.ipynb` — Main Jupyter notebook containing EDA, feature engineering, visualizations, and modeling pipelines.
- `climate_nasa.csv` — Social media / climate engagement dataset used for EDA and text/topic analysis.
- `Model_Data_Sets/` — Folder with datasets and generated submission files:
  - `train.csv` — Training data for emission prediction (contains target `emission`).
  - `test.csv` — Test set for which predictions are generated.
  - `sample_submission.csv` — Sample submission template.
  - `final_submission.csv` — Final saved predictions produced by the RandomForest pipeline.
  - `final_fast_submission.csv` — Final saved predictions produced by the HistGradientBoosting pipeline.

## Overview
This project performs two main tasks inside the notebook:
- EDA and behavioral analysis of NASA climate posts (time-series, engagement, topics, post length and timing).
- Emission prediction using tree-based regressors (RandomForestRegressor and HistGradientBoostingRegressor) trained on `Model_Data_Sets/train.csv` and used to create submissions saved into `Model_Data_Sets/`.

## Key Files Used in the Notebook
- Data ingestion: `climate_nasa.csv`, `Model_Data_Sets/train.csv`, `Model_Data_Sets/test.csv`.
- Modeling: features are derived from `train.csv`; models save outputs to `Model_Data_Sets/final_submission.csv` and `Model_Data_Sets/final_fast_submission.csv`.

## Dependencies
Suggested Python environment:

- Python 3.8+
- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn

Install with pip:

```
python -m venv .venv
.venv\Scripts\activate   # Windows
pip install pandas numpy matplotlib seaborn scikit-learn
```

Notes:
- `HistGradientBoostingRegressor` is available in recent `scikit-learn` versions; upgrade if necessary: `pip install -U scikit-learn`.

## How to run
1. Create and activate a Python virtual environment (see commands above).
2. Ensure all CSV files listed in the repository are present.
3. Open and run `Climate_Change_Modelling.ipynb` in JupyterLab / Jupyter Notebook, executing cells top-to-bottom. The notebook performs preprocessing, EDA, trains both models, and writes predictions to `Model_Data_Sets/`.

## Outputs
- `Model_Data_Sets/final_submission.csv` — generated by the RandomForest pipeline.
- `Model_Data_Sets/final_fast_submission.csv` — generated by the HistGradientBoosting pipeline.

## Notes & Recommendations
- The notebook contains both exploratory sections (engagement analysis) and modeling sections (emission prediction). If only the emission pipeline is required, run the cells under the model sections after confirming `train.csv` and `test.csv` locations.
- Consider creating a `requirements.txt` from your environment for exact package versions.
